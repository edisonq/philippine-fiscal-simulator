# Acceptable Use Policy

Philippine Fiscal Simulator is intended for education, research discussion, civic learning, scenario exploration, software development, and transparent public understanding of fiscal-policy trade-offs.

## You May Use This Project To

- learn about fiscal policy concepts
- test simplified economic scenarios
- teach students or the public
- build visualizations
- improve the code
- contribute better formulas, documentation, and data validation
- discuss policy trade-offs in a transparent way
- create derivative works that preserve the disclaimer and do not misrepresent the tool

## You May Not Use This Project To

- present simulator outputs as official government data
- claim the project produces guaranteed forecasts
- mislead the public, media, voters, investors, clients, or institutions
- impersonate a government agency or official research body
- provide legal, tax, financial, investment, or government advice without proper qualification
- remove or hide disclaimers when redistributing modified versions
- use the project for fraud, market manipulation, political deception, harassment, or unlawful activity
- blame the maintainers for decisions made using modified, incomplete, misunderstood, outdated, or misrepresented versions of the project

## Publishing Results

If you publish results from this simulator, clearly state:

1. the version used,
2. the data sources used,
3. the assumptions changed,
4. the limitations of the model,
5. that outputs are scenario estimates, not official predictions.

## Contributions and Moderation

The maintainers may reject contributions, issues, forks, links, or documentation that encourage misleading, harmful, unlawful, or deceptive use of the project.
