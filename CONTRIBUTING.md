# Contributing

Thanks for your interest in improving Philippine Fiscal Simulator.

## Good Contributions

Helpful contributions include:

- clearer documentation
- better formulas
- stronger disclaimers
- UI simplification
- accessibility fixes
- verified data-source notes
- tests and validation checks
- bug fixes
- localization
- game-mode ideas

## Contribution Rules

Please do not submit changes that:

- present outputs as official government forecasts
- remove disclaimers
- hide limitations
- add unsourced official-looking data
- encourage misleading or unlawful use

## Data Contributions

If contributing data, include:

- source owner
- source URL
- release date
- retrieval date
- unit basis
- transformation notes
- row status

Never mix partial-year data with complete annual data without clearly marking it.
