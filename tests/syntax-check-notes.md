# Syntax Check Notes

Suggested checks before pushing major changes:

1. Open `index.html` locally in a browser.
2. Check browser console for JavaScript errors.
3. Move every slider and confirm Household Reality updates immediately.
4. Confirm Sankey chart renders without `NaN` or `undefined` SVG values.
5. Click scenario tools: Save, Load, Set Base, Compare, JSON, CSV.
6. Click shocks: Rice Shock, Energy Shock, Recession, Clear.
7. Click forecasting tools with demo and imported CSV data.
